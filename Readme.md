## 应用简介

### 应用名称
中兴U30AIR状态监控

### 应用概述
中兴U30AIR状态监控是一款专为U30AIR 5G随身WiFi设备设计的软件，能够实时显示设备的各种状态信息，帮助用户轻松管理和监控网络连接。

### 主要功能
- **设备状态显示**: 实时显示U30AIR设备的各种状态信息，包括后台IP地址、开机时间、电量、充电状态、网络制式、信号强度等。
- **流量监控**: 实时监控和显示上行（上传）和下行（下载）的流量速率。当流量速率超过1024KB/s时，自动转换为MB/s显示，确保信息展示更加简洁和直观。
- **网络信息**: 显示当前网络制式和运营商信息，方便用户了解网络连接情况。
- **流量使用情况**: 记录并显示流量使用情况和剩余流量，帮助用户合理规划流量使用。
- **电量管理**: 实时显示电量百分比和充电状态，确保用户随时掌握设备的电量情况。

### 用户界面
- **简洁直观**: 界面设计简洁，所有信息一目了然，用户可以轻松查看和管理设备状态。
- **中文支持**: 完全支持中文显示，适合中国用户使用。

### 适用人群
- **随身WiFi用户**: 经常使用随身WiFi设备的用户，特别是中兴U30AIR 5G随身WiFi用户。
- **网络管理人员**: 需要随时了解设备状态和网络连接情况的网络管理人员。
